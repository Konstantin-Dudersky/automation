Модуль для подключения периферийных устройств по интерфейсу I²C.

Все каналы подключены через I²C-мультиплексор **TCA9548A** [^1], что позволяет подключать к разным каналам периферию с одинаковым адресом.

Все устройства гальванически изолированы от шины IBus с помощью чипа **CA-IS3021** [^2].

Подключённые устройства можно запитывать как 5В, так и 3.3В. Напряжение питания одинаковое для всех каналов.
