Представлена линейка модулей, по аналогии с модулями промышленных контроллеров.

Модули выполнены в одинаковом формате, имеют одинаковые внешние размеры и крепление. Связаны между собой 20-пиновым FFC-кабелем.

## Список модулей

Все модули можно разделить по группам:

- PMCP - модули контроллеров:
    - [PMCP-ESP32C3](/modules/pmcp/pmcp-esp32c3) - контроллер на базе микроконтроллера ESP32-C3
    - [PMCP-LLP](/modules/pmcp/pmcp-llp) - контроллер на базе одноплатника Luckfox Lyra
- PMIO - дискретные и аналоговые входы/выходы:
    - [PMIO-DIx32](/modules/pmio/pmio-dix32) - подключение 32 дискретных входов постоянного напряжения
    - [PMIO-DQx16](/modules/pmio/pmio-dqx16) - управление 16 дискретными транзисторными выходами
    - [PMIO-RQx8](/modules/pmio/pmio-rqx8) - управление 8 дискретными релейными выходами
- PMIF - интерфейсы:
- PMDR - драйверы двигателей:
- PMPS - блоки питания:
    - [PMPS-20W](/modules/pmps/pmps-20w) - блок питания с входом AC 230В
- PMLD - модули с светодиодной индикацией:
    - [PMLD-10](/modules/pmld/pmld-10) - 10 светодиодов
    - [PMLD-18](/modules/pmld/pmld-18) - 18 светодиодов

Размеры печатных плат модулей: 100 мм x 70 мм.

На всех печатных платах есть 4 отверстия для крепления стойками M4.

С одной стороны платы находится клеммник, который выходит за пределы печатной платы.

## Шина IBus

Модули подключаются друг к другу через 20-пиновый FFC-кабель. В этом кабеле передаётся:

- Постоянное напряжение нескольких номиналов: 24В, 5В, 3.3В.
- Шина I²C - для связи контроллера (PMCP) с модулями ввода/вывода (PMIO).
- Шина CAN - для связи контроллера (PMCP) с модулями ввода/вывода (PMIO). Линии CAN гальванически изолированы от остальной схемы.
- Сигнал RST - выход сторожевого таймера (watchdog) [STWD100](https://www.st.com/en/reset-and-supervisor-ics/stwd100.html). Необходим для сброса выходов при остановке программы управления.
