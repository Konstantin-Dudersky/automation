## Описание


- PMPS-xxx - модуль блока питания.
- PMCP-xxx - модули процессоров (Luckfox Lyra, ESP32).
- PMIO-xxx - модули преобразования электрических сигналов в цифровые и обратно. Дискретные и аналоговые входы/выходы.
- PMIF-xxx - модули интерфейсов (I²C, 1-Wire)

## Шина IBus

Модули подключаются друг к другу через 20-пиновый FFC-кабель. В этом кабеле передаётся:

- Постоянное напряжение нескольких номиналов: 24В, 5В, 3.3В. Источником является блок питания [PMPS-20W](/modules/pmps/pmps-20w)
- Шина I²C - для связи контроллера (PMCP) с модулями ввода/вывода (PMIO)
- Шина CAN - для связи контроллера (PMCP) с модулями ввода/вывода (PMIO)
- Сигнал RST
